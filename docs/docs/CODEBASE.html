Key Files

Screens

app/(tabs)/index.tsx

Dashboard screen showing balance summary, spending chart, and recent transactions. Uses react-native-chart-kit for visualization.

app/(tabs)/insights.tsx

AI-powered insights screen. Calls Groq API to generate personalized financial advice and health score.

app/(tabs)/reports.tsx

Financial reports with pie charts for category breakdown, bar charts for daily spending, and time range selection.

Services

lib/ai.ts

Groq API integration. Sends financial data to LLaMA 3.3 and parses AI response for insights, recommendations, and health score.

lib/database.ts

Firebase Realtime Database helpers. Functions for saving, retrieving, and subscribing to transactions, budgets, goals, and settings.

lib/notifications.ts

Expo Notifications implementation. Budget alerts, daily reminders, weekly summaries, and notification permission handling.

State Management

context/AppContext.tsx

React Context provider managing global state. Handles transactions, budgets, goals, user auth, currency, and dark mode preferences.

Configuration

app.json

Expo configuration including app name, icons, splash screen, Android package name, and Expo project ID.

eas.json

EAS Build profiles for development, preview (APK), and production (AAB) builds.
