Codebase Guide

Navigate the SpendWise project structure.

Project Structure

spendwisev2/
├── components/          # React components
│   ├── Auth.tsx         # Authentication UI
│   ├── Dashboard.tsx    # Main dashboard
│   ├── Transactions.tsx # Transaction list
│   ├── BudgetView.tsx   # Budget management
│   ├── Reports.tsx      # Analytics & charts
│   ├── Goals.tsx        # Savings goals
│   ├── Settings.tsx     # User settings
│   └── Layout.tsx       # App layout
├── context/
│   └── AppContext.tsx   # Global state
├── lib/
│   ├── auth.ts          # Firebase auth
│   └── firebase.ts      # Firebase config
├── docs/                # Documentation site
├── App.tsx              # Root component
└── vite.config.ts       # Vite configuration

Key Files

App.tsx

Main application component with routing logic and authentication state.

context/AppContext.tsx

Global state management using React Context for transactions, budgets, and goals.

lib/auth.ts

Firebase authentication functions including email/password and Google OAuth.
